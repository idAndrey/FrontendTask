Новые подходы к созданию резиновых макетов с помощью CSS Grid

10 июня 2024 Автор: Анна Смирнова Время чтения: 8 мин

Введение в CSS Grid

CSS Grid Layout — это мощная система для создания двумерных макетов, которая радикально изменила подход к вёрстке. В отличие от Flexbox, который работает в одном измерении, Grid позволяет точно контролировать и строки, и столбцы одновременно.

Базовый пример Grid-контейнера:
.container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 20px;
}

Резиновые макеты с auto-fit и minmax()

Ключевая особенность современных Grid-макетов — использование функции minmax() в сочетании с auto-fit. Это позволяет создавать по-настоящему гибкие сетки, которые адаптируются без медиазапросов.

Адаптивная сетка товаров:
.products-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 30px;
}

@media (max-width: 768px) {
  .products-grid {
    grid-template-columns: 1fr;
    gap: 15px;
  }
}

Именованные области

Grid позволяет создавать сложные макеты с именованными областями, что делает код более читаемым и поддерживаемым.

Макет страницы с именованными областями:
.page-layout {
  display: grid;
  grid-template-areas:
    "header header"
    "sidebar content"
    "footer footer";
  grid-template-columns: 250px 1fr;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.content { grid-area: content; }
.footer { grid-area: footer; }
